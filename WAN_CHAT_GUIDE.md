## 🚀 快速开始

### 方式 1: 云服务器（推荐）⭐

**在云服务器上**:

```bash
cd ~/.jvs/.openclaw/workspace/markcell-openclaw-AI-Chat
node wan-chat.js
```

**输出**:
```
========================================
🌐 OpenClaw 外网对话服务
========================================

📍 连接编码：OCLAW-XXXX-XXXX-XXXX
📍 公网 IP: 你的公网 IP
📍 外网地址：http://公网 IP:8080

✅ 外网对话服务已启动
```

**在本地电脑上**:

```bash
node examples/example-real-chat.js client <云服务器的编码> http://<云服务器公网 IP>:8080
```

---

### 方式 2: 家庭/办公室网络

#### 步骤 1: 获取公网 IP

**联系你的 ISP（网络服务提供商）**:
- 中国电信：10000
- 中国联通：10010
- 中国移动：10086

申请公网 IPv4 地址（部分地区的家庭宽带已不提供）

#### 步骤 2: 配置路由器端口转发

**登录路由器管理页面**:
1. 浏览器访问：`192.168.1.1` 或 `192.168.0.1`
2. 输入管理员账号密码
3. 找到"端口转发"或"虚拟服务器"
4. 添加规则：
   ```
   服务名称：OpenClaw
   外部端口：8080
   内部 IP: 这台电脑的局域网 IP（如 192.168.3.194）
   内部端口：8080
   协议：TCP
   状态：启用
   ```

#### 步骤 3: 开放防火墙

**macOS**:
```bash
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /usr/local/bin/node
```

**Linux**:
```bash
sudo ufw allow 8080/tcp
```

**云服务器（阿里云/腾讯云）**:
1. 登录控制台
2. 找到"安全组"
3. 添加入站规则：端口 8080

#### 步骤 4: 启动外网服务

```bash
node wan-chat.js
```

## 📋 外网对话配置

### 配置 1: 云服务器

**推荐服务商**:
- 阿里云 ECS
- 腾讯云 CVM
- 华为云 ECS
- AWS EC2

**配置步骤**:
```bash

# 2. 配置安全组（在云控制台）
# 添加入站规则：端口 8080

# 3. 启动服务

```

**优点**:
- ✅ 自带公网 IP

- ✅ 稳定可靠
- ✅ 配置简单

**成本**: 约 60-100 元/月（基础配置）

---

### 配置 2: 家庭网络

**要求**:
- ✅ 公网 IPv4 地址
- ✅ 可配置的路由器
- ✅ 电脑常开

**步骤**:
```bash
# 1. 确认有公网 IP
curl ifconfig.me
# 应该返回公网 IP（不是 10.x.x.x, 172.16.x.x, 192.168.x.x）

# 2. 配置路由器端口转发
# 登录路由器 → 端口转发 → 添加规则

# 3. 启动服务
node wan-chat.js
```

**优点**:
- ✅ 免费
- ✅ 控制在自己手中

**缺点**:
- ❌ 需要公网 IP（部分地区已不提供）
- ❌ 需要电脑常开
- ❌ 配置较复杂

---

### 配置 3: 内网穿透

**使用 ngrok**:
```bash
# 1. 安装 ngrok
# 访问 https://ngrok.com 下载

# 2. 启动本地服务
node wan-chat.js

# 3. 在另一个终端启动 ngrok
ngrok http 8080

# 4. 获得公网地址
# 如：https://abc123.ngrok.io
```

**使用 frp（自建）**:
```bash
# 1. 在云服务器上安装 frps
# 2. 在本地安装 frpc
# 3. 配置 frpc.ini
[openclaw]
type = tcp
local_ip = 127.0.0.1
local_port = 8080
remote_port = 8080

# 4. 启动 frpc
frpc -c frpc.ini
```

**优点**:
- ✅ 不需要公网 IP
- ✅ 配置相对简单

**缺点**:
- ❌ 可能需要费用（ngrok 免费版有限制）
- ❌ 依赖第三方服务

## 🔧 故障排查

### 问题 1: 无法获取公网 IP

**症状**: `公网 IP: 未获取到`

**解决方案**:
```bash
# 手动获取
curl ifconfig.me
curl api.ipify.org

# 如果返回的是 10.x.x.x, 172.16.x.x, 192.168.x.x
# 说明你在内网，需要联系 ISP 获取公网 IP
```

### 问题 2: 连接超时

**症状**: `ETIMEDOUT`

**解决方案**:
1. 检查端口转发是否配置正确
2. 检查防火墙是否开放 8080 端口
3. 确认公网 IP 是否正确
4. 测试本地访问：`curl http://localhost:8080/status`

### 问题 3: 连接被拒绝

**症状**: `Connection refused`

**解决方案**:
1. 确认服务是否运行：`netstat -an | grep 8080`
2. 确认监听地址：应该是 `0.0.0.0:8080`
3. 检查防火墙设置
