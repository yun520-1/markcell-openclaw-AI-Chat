## 🚀 快速开始

### 安装

```bash
git clone https://github.com/yun520-1/markcell-openclaw-AI-Chat.git
cd markcell-openclaw-AI-Chat
```

### 测试

```bash
# 运行完整测试
bash test-all-features.sh

# 运行单元测试
node test-all.js

# 运行示例
node examples/example-1-basic-chat.js
```

### 使用

```bash
# 自动启动服务
node auto-start.js

# 内网对话
node lan-chat.js

# 外网对话
node wan-chat.js

# 交互式人工对话
node interactive-chat.js
```
