## 🚀 快速使用

### 安装

```bash
git clone https://github.com/yun520-1/markcell-openclaw-AI-Chat.git
cd markcell-openclaw-AI-Chat
```

### 测试

```bash
# 完整功能测试
bash test-all-features.sh

# 单元测试
node test-all.js
```

### 使用

```bash
# 交互式人工对话
node interactive-chat.js

# 内网对话
node lan-chat.js

# 外网对话
node wan-chat.js

# 自动启动
node auto-start.js
```
