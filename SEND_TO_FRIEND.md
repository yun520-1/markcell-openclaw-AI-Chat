# 🌐 OpenClaw 外网服务配置指南

## 👋 你好！请按以下步骤配置外网服务：

---

### 1️⃣ 启动外网对话服务

```bash
cd ~/.qclaw/workspace/skills/markcell-openclaw-ai-chat
node wan-chat.js
```

**启动后应该看到**:
```
========================================
🌐 OpenClaw 外网对话服务
========================================

📍 连接编码：OCLAW-CECB-2C97-44F3
📍 公网 IP: [你的公网 IP]
📍 端口：1238
📍 外网地址：http://[你的公网 IP]:1238

✅ 外网对话服务已启动，等待连接...
```

**⚠️ 重要**: 保持这个终端运行，不要关闭！

---

### 2️⃣ 获取你的公网 IP

在**另一个终端**执行：

```bash
curl ifconfig.me
```

**记下输出的 IP 地址**（例如：49.87.224.177）

---

### 3️⃣ 配置防火墙（如果在云服务器上）

#### 阿里云
1. 登录阿里云控制台
2. 进入 **云服务器 ECS** → **安全组**
3. 添加 **入站规则**:
   - 协议：TCP
   - 端口：**1238**
   - 授权对象：0.0.0.0/0

#### 腾讯云
1. 登录腾讯云控制台
2. 进入 **云服务器** → **安全组**
3. 添加 **入站规则**:
   - 协议端口：TCP:1238
   - 来源：0.0.0.0/0

#### 华为云
1. 登录华为云控制台
2. 进入 **弹性云服务器** → **安全组**
3. 添加 **入站规则**:
   - 协议/端口：TCP:1238
   - 源地址：0.0.0.0/0

---

### 4️⃣ 测试服务

在**另一个终端**执行：

```bash
curl http://localhost:1238/status
```

**应该返回 JSON 格式的服务器状态信息**

## 🚨 常见问题

### 问题 1: 端口被占用

**错误**: `Error: listen EADDRINUSE`

**解决**:
```bash
# 找到占用进程
lsof -i :1238

# 杀死进程
kill -9 [PID]

# 重新启动
node wan-chat.js
```

### 问题 2: 无法获取公网 IP

**解决**: 尝试其他服务
```bash
curl api.ipify.org
curl ifconfig.co
```

### 问题 3: 外网无法访问

**检查**:
1. 安全组是否配置正确
2. 端口是否是 1238
3. 服务是否正在运行
