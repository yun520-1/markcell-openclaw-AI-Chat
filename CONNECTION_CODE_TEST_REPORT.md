## 💡 解决方案

### 方案 1: 手动交换地址（最简单）

双方互相告知自己的服务器地址：

**你发送给对方**:
```
我的连接编码：OCLAW-621E-28C1-4CBA
我的服务器地址：http://你的公网 IP:8080
```

**对方回复**:
```
我的连接编码：OCLAW-58F8-1FE2-CE27
我的服务器地址：http://对方的公网 IP:8080
```

**然后发送消息**:
```javascript
await server.sendToRemote('http://对方的公网 IP:8080/message', {
  from: 'OCLAW-621E-28C1-4CBA',
  content: '你叫什么名字？',
  type: 'message'
});
```

### 方案 2: 中央注册表服务

创建一个中央服务来映射连接编码到网络地址：

```javascript
// 注册表服务器
const registry = new Map();

// 注册编码
registry.set('OCLAW-58F8-1FE2-CE27', {
  code: 'OCLAW-58F8-1FE2-CE27',
  address: 'http://1.2.3.4:8080',
  registeredAt: Date.now()
});

// 查找地址
function lookupAddress(code) {
  return registry.get(code);
}
```

### 方案 3: 使用现有通信渠道

通过其他方式（微信、邮件等）先交换地址信息，然后使用连接编码进行后续通信。

## 🚀 实际操作步骤

### 步骤 1: 获取你的公网 IP

```bash
# macOS/Linux
curl ifconfig.me

# 或访问 https://ifconfig.me
```

假设你的公网 IP 是：`123.45.67.89`

### 步骤 2: 启动服务器并告知对方

```javascript
const { NetworkServer } = require('./modules/a2a/network_server');

const server = new NetworkServer({
  port: 8080,
  host: '0.0.0.0',  // 监听所有网络接口
  connectionCode: 'OCLAW-621E-28C1-4CBA'
});

await server.start();

```

然后告诉对方：
```
我的连接编码：OCLAW-621E-28C1-4CBA
我的服务器地址：http://123.45.67.89:8080
```

### 步骤 3: 对方发送消息

对方收到你的地址后，可以发送消息：

```javascript
await server.sendToRemote('http://123.45.67.89:8080/message', {
  from: 'OCLAW-58F8-1FE2-CE27',
  content: '你好！',
  type: 'message'
});
```

### 步骤 4: 接收并回复

你的服务器会收到消息：

```javascript
server.on('message', (msg) => {

  // 回复
  server.sendToRemote('http://对方的 IP:8080/message', {
    from: 'OCLAW-621E-28C1-4CBA',
    content: '你好！收到你的消息了！',
    type: 'reply'
  });
```

## 📖 完整示例

### 服务器端代码

```javascript
const { NetworkServer } = require('./modules/a2a/network_server');

async function startServer() {
  const server = new NetworkServer({
    port: 8080,
    host: '0.0.0.0',
    connectionCode: 'OCLAW-621E-28C1-4CBA'
  });
  
  await server.start();
  console.log('服务器已启动：http://0.0.0.0:8080');
  
  // 监听消息
  server.on('message', async (msg) => {
    console.log(`\n📥 收到消息:`);
    console.log(`来自：${msg.from}`);
    console.log(`内容：${msg.content}`);
    
    // 这里可以添加自动回复逻辑
    if (msg.content.includes('你叫什么名字')) {
      console.log('\n📤 自动回复...');
      // 需要知道对方的地址才能回复
    }
  });
  
  // 保持服务器运行
  console.log('\n等待消息中...\n');
}

startServer();
```
