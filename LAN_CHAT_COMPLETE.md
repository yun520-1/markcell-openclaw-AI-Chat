## 🚀 使用方法

### 方式 1: 两台设备对话（推荐）

**在设备 A 上**:
```bash
cd ~/.jvs/.openclaw/workspace/openclaw-dialog-tools
node lan-chat.js
```

**在设备 B 上**（同一局域网）:
```bash
cd ~/.jvs/.openclaw/workspace/openclaw-dialog-tools
node lan-chat.js
```

现在两台设备可以互相发送消息并自动回复！

## 🔧 配置选项

### 修改端口

编辑 `lan-chat.js`:
```javascript
this.server = new NetworkServer({
  port: 8080,  // 修改这里
  host: '0.0.0.0'
});
```

### 自定义回复

编辑 `generateReply` 函数:
```javascript
generateReply(content, fromCode) {
  if (content.includes('自定义')) {
    return '自定义回复';
  }
  // ...
}
```
