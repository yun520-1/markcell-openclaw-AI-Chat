```bash

```

### 方式 2: 使用 curl

```bash
# 发送消息
curl -X POST http://localhost:1234/message \
  -H "Content-Type: application/json" \
  -d '{
    "from": "test",
    "fromCode": "OCLAW-TEST",
    "content": "你好",
    "type": "message",
    "timestamp": '$(date +%s)'
```
