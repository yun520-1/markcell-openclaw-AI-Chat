## 💡 解决方案

### 如果目标是另一台设备

**在目标设备上执行**:

1. **启动内网服务**
   ```bash
   node lan-start.js
   ```

2. **确认自动回复**
   - 确保 `lan-start.js` 中的 `autoReply` 函数正常工作

3. **检查防火墙**
   ```bash
   # macOS
   sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /usr/local/bin/node
   ```

### 如果目标是本机

**问题**: 发送和接收都在同一台机器上，可能导致：
- 端口冲突
- 自动回复逻辑未触发

**解决方案**:
1. 在目标设备上运行内网服务
2. 使用不同的设备进行测试
