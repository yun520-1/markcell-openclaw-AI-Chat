# 📡 信令服务器使用指南

## 🎯 什么是信令服务器？

信令服务器是一个**轻量级协助服务器**，帮助双方：
1. 交换连接信息（IP、端口、编码）
2. 在 P2P 连接失败时中转消息
3. 不存储聊天记录，完全去中心化

## 🔍 故障排查

### Q1: 无法连接信令服务器

**症状**: `❌ 启动失败：connect ECONNREFUSED`

**解决方案**:
1. 确认信令服务器已启动
2. 确认 IP 地址和端口正确
3. 检查防火墙是否开放 8080 端口
4. 测试：`telnet 49.87.224.177 8080`

### Q2: 找不到对方

**症状**: `❌ 错误：对方不在线`

**解决方案**:
1. 确认对方已连接信令服务器
2. 确认编码输入正确
3. 检查对方是否断开连接

### Q3: 消息发送失败

**症状**: 消息发不出去

**解决方案**:
1. 确认已创建/加入房间
2. 确认与信令服务器连接正常
3. 查看 `/history` 确认消息状态
