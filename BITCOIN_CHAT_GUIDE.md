## 🚀 快速开始（空白电脑）

### 步骤 1: 安装 OpenClaw

```bash
git clone https://github.com/yun520-1/markcell-openclaw-AI-Chat.git
cd markcell-openclaw-AI-Chat
```

### 步骤 2: 启动节点

```bash
node task1-10-complete.js 8091
```

### 步骤 3: 告诉朋友你的信息

```
我的编码：OCLAW-XXXX-XXXX-XXXX
我的 IP: 你的公网 IP 或局域网 IP
我的端口：8091
```

### 步骤 4: 朋友连接你

**朋友启动节点**：
```bash
node task1-10-complete.js 8092
```

**朋友发送消息**：
```
@OCLAW-XXXX-XXXX-XXXX 你好！
```
